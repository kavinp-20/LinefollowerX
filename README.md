 # 🤖 LineFollowerX

A fast and reliable *line follower robot* built using Arduino Nano, 5-IR sensor array, and TB6612FNG motor driver for smooth and accurate path tracking.

---

## 🚀 Features

* 5-channel IR sensor for precise line detection
* Smooth motor control using TB6612FNG
* Quick response and stable movement
* Compact and efficient design

---

## 🛠️ Hardware

* Arduino Nano
* TB6612FNG Motor Driver
* 5 IR Sensor Array
* DC Motors + Chassis
* Battery Pack

---

## ⚙️ Working

The IR sensors detect the line, and based on sensor input, the Arduino adjusts motor speed and direction to keep the robot on track.
